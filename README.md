## Overview

**Zyntax Studio** is a modern, high-performance IDE built specifically to supercharge your **Zyntax** development workflow. Powered by **Electron** and the logic of the **Monaco Editor** (VS Code engine), it delivers a premium, native coding experience.

Unlike generic text editors, Zyntax Studio understands your code. From intelligent syntax highlighting to direct script execution, every feature is designed to make writing Zyntax faster and error-free.

## Key Features

### Intelligent Editing
- **Monaco Engine**: Experience the same fluid editing as VS Code.
- **Zyntax Syntax Highlighting**: Color-coded tokens for keywords (`say`, `if`, `function`), strings, and logic.
- **Smart Brackets**: Auto-closing brackets and quotes.
- **Minimap**: Visual overview of your entire codebase.

### Power Tools
- **Integrated Terminal**: Execute `run` commands and scripts directly without leaving the IDE.
- **Command Palette** (`Ctrl+Shift+P`): Access every functionality via the keyboard.
- **File Explorer**: Drag-and-drop file management, folder creation, and workspace organization.

### Developer Experience
- **Git Integration**: View changes, stage files, commit, and push directly from the sidebar.
- **Customizable**: Tweak settings for font size, theme, and editor behavior.
- **Project Workspaces**: Open folders as projects with persistent state.

## Installation

### Windows
1. Download the latest `.exe` from the [Releases Page](https://github.com/ZyntaxFoundation/zyntax-studio/releases).
2. Run the installer.
3. Start coding in Zyntax!
