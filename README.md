## Overview

**Zyntax Studio** is a modern, high-performance Integrated Development Environment built specifically for Zyntax. Powered by **Electron** and the **Monaco Editor** (the heart of VS Code), it delivers a premium coding experience with native performance.

## Features

- **Monaco Editor Engine**: Full IntelliSense, syntax highlighting, and code folding.
- **Integrated Terminal**: Run Zyntax scripts directly within the IDE.
- **Smart File Explorer**: Drag-and-drop support, project workspaces, and file icons.
- **VS Code Aesthetics**: Familiar Dark+ theme and layout for a seamless transition.
- **Zyntax Native**: Deep integration with the Zyntax toolchain (ZPM, Compiler).
